# 极简库存管理系统 - 需求规格说明书

## 1. 项目概述
本系统为小微企业/小型仓库单机轻量化库存管理工具，摒弃复杂的供应链、多终端、多级审批功能，仅聚焦商品入库、出库、库存查询、台账追溯核心场景，替代传统手工记账，实现库存数据规范化、精准化管理，适配1-3人小型团队单机办公使用。

## 2. 用户角色
为简化架构，仅设置两类角色，权限清晰无冗余：
- **管理员**：全功能权限，包含商品管理、出入库操作、台账查询、报表导出、账号管理、数据备份
- **普通操作员**：仅基础操作权限，可执行入库、出库、库存查询、个人操作记录查看，无账号管理、数据修改、报表导出权限

## 3. 核心功能需求

### 3.1 商品基础管理
- 支持商品新增、编辑、删除、名称模糊查询
- 商品核心字段：名称、规格、单位、初始库存、备注
- 仅可删除无库存、无操作记录的闲置商品

### 3.2 入库管理
- 支持手动录入入库信息（商品、数量、可选备注）
- 提交后自动更新库存
- 可按时间查看全部入库记录，信息全程留存可追溯

### 3.3 出库管理
- 支持手动出库操作
- 系统自动校验库存，禁止负库存出库并弹窗提示
- 可查询所有出库记录，展示商品、数量、操作人、时间等核心信息

### 3.4 库存查询
- 实时展示全部商品当前库存数据
- 支持按商品名称精准搜索
- 出入库操作完成后，库存数据即时刷新

### 3.5 台账与简易报表
- 整合所有出入库流水，按时间倒序展示
- 支持按日期区间筛选记录
- 可导出Excel格式台账报表

### 3.6 账号密码管理
- 管理员可新增、删除普通操作员账号
- 所有用户可自主修改登录密码
- 系统需账号密码登录
- 系统初始化时创建默认管理员账号（用户名：admin，密码：admin123），首次登录后应强制修改密码

## 4. 推荐实现方案

### 4.1 产品形态与部署方式
- 采用Windows桌面单机软件（EXE安装包）形式
- 无需搭建云端服务器、无需部署内网服务
- 支持离线运行

### 4.2 数据存储方案
- 数据全部本地存储于电脑本地硬盘
- 采用轻量化嵌入式SQLite数据库
- 支持手动数据备份

### 4.3 运行环境
- 适配主流Windows系统
- 普通办公电脑即可流畅运行

## 5. 非功能需求
- **性能**：轻量化运行，操作响应速度≤1秒，支持日常常规数据量存储与查询
- **易用性**：界面简洁、操作简单，单次出入库操作≤3步，关键操作有结果弹窗提示
- **数据安全**：所有操作全程留痕，历史流水记录不可删除、不可修改

## 6. 明确不实现功能
- 不支持多仓库、多门店、仓位管理；不支持多设备云端同步
- 不做供应商、客户信息管理模块
- 无库存预警、智能补货、数据可视化统计等高级功能
- 无批量操作、批量导入导出功能
- 无移动端、网页端，仅保留单机桌面端
- 无多级审批、复杂权限分级体系

## 7. 验收标准

### 7.1 功能验收
- **商品管理**: 新增、编辑、删除、模糊查询功能正常；仅可删除无库存、无操作记录的商品
- **入库管理**: 手动录入入库信息正常，库存自动更新；入库记录可按时间查看、追溯
- **出库管理**: 手动出库操作正常，系统自动校验库存，负库存出库被拦截并弹窗提示；出库记录可查询
- **库存查询**: 实时展示全部商品库存，支持名称精准搜索，数据即时刷新
- **台账报表**: 按时间倒序展示出入库流水，支持日期区间筛选，Excel导出功能正常
- **账号管理**: 管理员可增删普通操作员账号；所有用户可修改密码；系统需账号密码登录

### 7.2 性能验收
- 操作响应速度≤1秒
- 支持至少1000种商品、10000条操作记录的存储与查询
- 系统启动时间≤3秒

### 7.3 数据安全验收
- 所有操作全程留痕，历史流水记录不可删除、不可修改
- 数据本地存储正常，备份与恢复功能可用
- 权限控制准确：普通操作员无账号管理、数据修改、报表导出权限

### 7.4 兼容性验收
- 适配Windows 10及以上版本（32位/64位）
- 在普通办公电脑（CPU i3及以上、内存4GB及以上）上流畅运行

### 7.5 易用性验收
- 界面简洁，操作简单，单次出入库操作≤3步
- 关键操作有结果弹窗提示（成功/警告/错误）
- 无卡顿、报错、闪退等问题
