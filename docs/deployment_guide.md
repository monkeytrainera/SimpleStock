# 极简库存管理系统 - 部署指南

## 1. 系统要求

### 1.1 硬件要求
| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | Intel Core i3 或同等处理器 | Intel Core i5 或更高 |
| 内存 | 4GB RAM | 8GB RAM |
| 磁盘空间 | 100MB 可用空间 | 500MB 可用空间 |
| 显示器 | 1024x768 分辨率 | 1920x1080 或更高 |

### 1.2 软件要求
| 项目 | 版本要求 |
|------|----------|
| 操作系统 | Windows 10 及以上（32位/64位） |
| .NET Framework | 4.0 及以上（通常已预装） |

---

## 2. 安装指南

### 2.1 获取安装包
- 从官方渠道获取 `SimpleStock_Setup.exe` 安装包
- 验证安装包完整性（可选）

### 2.2 安装步骤

#### 步骤1：运行安装程序
1. 双击 `SimpleStock_Setup.exe` 启动安装向导
2. 如果出现用户账户控制提示，点击"是"允许安装

#### 步骤2：欢迎界面
- 点击"下一步"继续安装

#### 步骤3：许可协议
- 阅读许可协议
- 勾选"我接受许可协议"
- 点击"下一步"

#### 步骤4：选择安装位置
- 默认安装路径：`C:\Program Files\SimpleStock\`
- 可点击"浏览"选择其他安装位置
- 点击"下一步"

#### 步骤5：开始菜单快捷方式
- 选择是否创建开始菜单快捷方式
- 点击"下一步"

#### 步骤6：确认安装
- 确认安装设置
- 点击"安装"开始安装

#### 步骤7：安装完成
- 安装进度完成后，点击"完成"退出安装向导
- 可选择是否立即启动应用

## 3. 首次启动与配置

### 3.1 首次启动
1. 从桌面快捷方式或开始菜单启动应用
2. 系统会自动创建数据库文件
3. 显示登录界面

### 3.2 默认管理员账号
- **用户名**：`admin`
- **密码**：`admin123`

### 3.3 首次登录流程
1. 输入用户名 `admin` 和密码 `admin123`
2. 点击"登录"按钮
3. 系统会提示强制修改密码
4. 设置新密码后进入主界面

## 4. 数据备份与恢复

### 4.1 数据备份

#### 方法1：通过软件备份
1. 登录系统
2. 点击顶部菜单栏"文件" → "数据备份"
3. 选择备份保存路径（默认：`安装目录\backup\`）
4. 点击"确认"完成备份

#### 方法2：手动备份
1. 关闭应用程序
2. 找到数据库文件：`安装目录\data\simple_stock.db`
3. 复制数据库文件到安全位置

### 4.2 数据恢复

#### 方法1：通过软件恢复（推荐）
1. 关闭应用程序
2. 将备份文件解压到 `安装目录\backup\` 目录
3. 启动应用
4. 点击顶部菜单栏"文件" → "数据恢复"
5. 选择备份文件
6. 点击"确认"完成恢复

#### 方法2：手动恢复
1. 关闭应用程序
2. 将备份的 `.db` 文件复制到 `安装目录\data\` 目录
3. 覆盖原有文件（建议先备份当前文件）
4. 启动应用

## 5. 卸载指南

### 5.1 通过控制面板卸载
1. 打开"控制面板" → "程序和功能"
2. 在程序列表中找到"极简库存管理系统"
3. 右键点击选择"卸载"
4. 按照卸载向导完成卸载

### 5.2 注意事项
- 卸载前请确保已备份数据
- 卸载后数据库文件将被删除
- 桌面快捷方式和开始菜单条目将被移除

## 6. 常见问题与解决方案

### 6.1 安装问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 安装程序无法启动 | 权限不足 | 右键点击安装程序，选择"以管理员身份运行" |
| 安装失败提示缺少组件 | .NET Framework未安装 | 安装 .NET Framework 4.0 或更高版本 |
| 安装路径权限不足 | 目标文件夹只读 | 选择其他安装路径或以管理员身份安装 |

### 6.2 运行问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 应用无法启动 | 数据库文件损坏 | 从备份恢复数据库或重新安装 |
| 登录失败 | 用户名或密码错误 | 检查输入，管理员可重置密码 |
| 界面显示异常 | 分辨率不兼容 | 调整屏幕分辨率至1024x768以上 |
| 操作响应慢 | 数据量过大 | 定期清理历史记录或增加内存 |

### 6.3 数据问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 库存数据错误 | 异常断电或系统崩溃 | 从备份恢复数据 |
| 导出Excel失败 | 磁盘空间不足 | 清理磁盘空间或选择其他保存位置 |
| 备份失败 | 目标路径不可写 | 选择可写路径或以管理员身份运行 |

## 9. 配置说明

### 9.1 应用目录结构
```
SimpleStock/
├── SimpleStock.exe          # 主程序
├── data/                    # 数据目录
│   └── simple_stock.db      # SQLite数据库文件
├── backup/                  # 备份目录
├── logs/                    # 日志目录
│   └── app.log              # 应用日志
└── config/                  # 配置目录
    └── app.ini              # 应用配置
```

### 9.2 配置文件说明
配置文件 `config/app.ini` 包含以下可配置项：
- `backup_path`: 备份文件保存路径
- `log_level`: 日志级别（DEBUG/INFO/WARNING/ERROR）
- `auto_backup_remind`: 是否提醒备份（true/false）

### 9.3 修改配置
1. 关闭应用程序
2. 使用文本编辑器打开 `config/app.ini`
3. 修改配置项
4. 保存文件
5. 重新启动应用
